# 優先順位をつけて対応しよう

ブログサイトを自作していますが、やらないといけないこと、やりたいこと、できることがごちゃごちゃで整理するために優先順位をつけて実施していきます。

## 1. やらないといけないこと

ブログサイトを公開するためにやらないといけないことをリストにします。
仕事の場合は、ここに番号、期限、担当者などを入ればWBSになります。
ここは、自分で使うだけだので、メモ程度です。  

| 項目 | 手段 |
| --- | --- |
| ブログ記事の体裁を整える | HTMLタグごとにCSSの値を決める |
| ブログ記事の中身を校正する | Markdown、国語 |
| ブログ一覧のソーティング | graphqlのソート |
| ブログ一覧の体裁を整える | HTMLタグごとにCSSの値を決める |
| SEO対策する | Gatsbyのプラグインを実装 |
| アフィリエイトを埋め込む | Webサイトを参照、Gatsbyのプラグインを実装 |

## 2. やりたいこと

勉強のためやブログサイトをかっこよくするためにやりたいことをリストにします。  

| 項目 | 手段 |
| --- | --- |
| PC用とモバイル用で表示を変える | CSS |
| いい感じのヘッダにする (一番上で止まって、下だけスクロール) | CSSの実装 |
| 有名なキーワードに勝手にリンクを貼る | やり方不明 |
| こどもの算数ができるようになるアプリを作る | Reactでのやり方を勉強して実装 |
| ブログ記事タイトルの左側にアイコンを付ける | CSS |
